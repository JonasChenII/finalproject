About this project

NYC Asian Eats is a three-page web project designed to showcase clear structure, responsive layout, and simple interaction. The site begins with an introduction, continues with a cuisine selection page, and ends with detailed restaurant recommendations including addresses and tips.

The goal is not to list every restaurant, but to highlight a few strong choices for each cuisine and help users decide where to eat more easily.

The background audio is available on all pages and restarts when navigating between pages, which is expected behavior for a multi-page site.
